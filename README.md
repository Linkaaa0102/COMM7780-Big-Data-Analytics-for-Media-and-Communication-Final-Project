# Video Generation Prompt Optimization System

## Overview

This project is an AIGC-oriented video prompt optimization system.

The system automatically converts simple user keywords or creative descriptions into structured and high-quality video-generation prompts for platforms such as Jimeng and Seedance.

The main goal of the project is to reduce the difficulty of prompt writing, improve video-generation efficiency, and optimize the AIGC video creation workflow.

---

## Core Features

### 1. Prompt Optimization

The system supports:

- Intelligent keyword analysis
- Multi-dimensional knowledge enhancement
- Style enhancement
- Platform adaptation
- Automatic negative prompt recommendation

Users only need to input a simple description to generate a complete and professional video-generation prompt.

---

### 2. Knowledge Base Management

The system includes an internal prompt knowledge base.

Functions include:

- Knowledge browsing
- Content searching
- Platform filtering
- Category filtering
- Professional terminology learning

---

## Supported Platforms

- General video-generation platforms
- Jimeng
- Seedance

Different prompt styles can be generated according to platform requirements.

## Usage

### System Startup

Double-click:

```text
启动系统.bat
```

The browser will automatically open after startup.

If it does not open automatically, visit:

```text
http://localhost:8000
```

---

### Workflow

1. Enter creative description
2. Select target platform
3. Select style preference
4. Click “Generate Optimization Suggestion”
5. Get optimized prompt
6. Copy and use the prompt
